医疗健康信息检索 Demo

基于关键词匹配和结构化结果展示的健康信息检索前端项目。

本页面用于展示基础文本处理、症状关键词匹配、健康信息检索和风险提醒逻辑。该 Demo 不提供医学诊断、处方或治疗方案，不能替代医生或专业医疗机构。

前端规则模拟

无需后端、无需 API Key，可直接部署到 GitHub Pages。

  • 关键词匹配
  • 风险提醒
  • 结构化结果

Interactive Demo

Demo 输入区

填写或选择一个样例，系统会基于本地健康知识库进行关键词匹配和风险信号检测。

输入配置

是否有高风险信号

示例输入

点击样例按钮可自动填充表单，方便快速测试。

结果模块

结构化展示匹配关键词、相关健康信息、风险等级和下一步动作。

等待输入

填写任意信息后点击“分析”，这里会显示结构化结果。

历史记录

历史记录保存在浏览器 localStorage 中，刷新页面后仍会保留。

Evaluation

测试说明

  • 自建测试样例：20 条
  • 关键词匹配准确率：约 80%
  • 高风险提示覆盖：胸痛、呼吸困难、意识异常、持续高热、严重过敏
  • 测试结果仅用于说明 Demo 的文本匹配效果，不代表医学准确性

Safety Boundary

安全边界说明

本项目为公开前端 Demo，仅用于展示文本处理、关键词匹配、健康信息检索和风险提醒。页面内容不构成医学诊断、处方或治疗建议，不能替代医生或专业医疗机构。若出现胸痛、呼吸困难、意识异常、严重过敏、持续高热等高风险情况，应及时联系医生或当地急救服务。
